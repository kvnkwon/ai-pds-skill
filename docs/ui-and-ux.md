# UI and UX

## Core Layout

Left: Chat
Right: Sections

## Behavior

- chat drives thinking
- sections show structure
- system suggests updates

## Key Elements

- stage progress bar
- section status (locked, active, complete)
- inline save prompts
- guided options

## UX Principles

- no blank page
- no overwhelming UI
- progressive reveal
- fast interactions

## Key Feeling

User should feel:
"I am being guided"

Not:
"I am filling out a form"
