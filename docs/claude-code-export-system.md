# Claude Code Export System

## Purpose

Bridge thinking → building

## Output Types

1. product-context.md
2. claude-code-starter.md

## Philosophy

Export:
- context
- decisions
- priorities
- constraints

Do NOT export:
- raw chat
- everything discussed

## Technical Approach

Support 3 modes:

- Open (AI decides stack)
- Preferred (user leans)
- Locked (explicit stack)

## Key Feature

User selects:
"How technical should this be?"

## Goal

Give Claude Code:
- enough context
- enough flexibility
- clear priorities
