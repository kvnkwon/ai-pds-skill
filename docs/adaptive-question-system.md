# Adaptive Question System

## Core Principle

Do not hardcode questions.
Hardcode required clarity.

## Question Types

1. Multiple choice (default)
2. Clarifying question
3. Draft + confirm

## Pattern

AI decides:
- what is missing
- what question type to use

## Example

Instead of:
"What is your target user?"

Use:
"Who is this mainly for?"

Options:
- beginners
- professionals
- businesses
- not sure
- custom

## Rules

- max 3 to 5 options
- must be distinct
- must be high quality
- must reduce thinking effort

## Always include

Custom option:
"Something else"

## Special Feature

"I’m stuck"

Generates:
- 3 smart questions
- with options

## Goal

Remove prompt paralysis
Guide thinking
Surface blind spots
