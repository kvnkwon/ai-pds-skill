# Core Product Concept

## What This Is

A structured AI workspace that simulates a design sprint.

## What This Is Not

- Not a chatbot
- Not a note-taking app
- Not a brainstorming tool

## Core Model

Project = folder
Sections = structured documents (like .md files)
Chat = thinking layer
AI = facilitator + expert + synthesizer

## Core Loop

1. User chats
2. AI extracts insights
3. AI proposes structured artifacts
4. User confirms
5. System updates sections
6. System maintains summary

## Core Principles

- structure over chat
- artifacts over messages
- progression over dumping
- clarity over speed
- guidance over freedom (but still flexible)

## Key Differentiator

Chat produces proposals
Workspace stores decisions

## Outcome

A persistent, evolving product brain
