# File System and Exports

## Model

Project = folder
Sections = markdown files

## Example

/project-name
- problem-statement.md
- jtbd.md
- mvp.md
- etc

## Export Options

- markdown (.md)
- PDF
- docx

## Export Types

1. Single file
2. Full project (zip)
3. Combined doc

## Purpose

Align with:
- developer workflows
- AI workflows
- structured thinking

## UX

Do not show file system early
Reveal gradually as project matures
