# Target User

## Primary Persona

AI-first builder

## Characteristics

- Non-technical or semi-technical
- Not product-trained
- Uses AI tools heavily
- Idea-driven
- Action-oriented
- Impatient
- Wants fast results

## Tools They Use

- ChatGPT
- Claude
- Cursor
- Replit
- Bolt.new

## Behaviors

- Jumps into building quickly
- Skips planning
- Relies on AI for everything
- Iterates chaotically

## Pain Points

- Doesn't know what to build
- Doesn't know if idea is good
- Gets stuck mid-build
- Restarts frequently
- Builds wrong thing
- Lacks structure

## Core Need

"I can build anything, but I don’t know what I should build."

## Desired Outcome

- clarity
- confidence
- direction
- structured plan
- fast path to building

## Secondary Personas

- Product designers exploring ideas
- PMs validating concepts
- Indie hackers
- Early-stage founders
